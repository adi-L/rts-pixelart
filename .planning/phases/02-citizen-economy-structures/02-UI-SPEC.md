# Phase 2 — UI Design Contract

> Visual and interaction contract for Citizen Economy & Structures. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Note:** This is a Phaser 3 canvas game. All visual values are in-engine constants (pixels, hex colors, tween configs), not CSS/DOM tokens. There is no web UI framework, no shadcn, no Tailwind.

## HUD Contract (COIN-03, COIN-04, D-14, D-15, D-16)

### Layout

Position: **top-left**, fixed to camera (`setScrollFactor(0)`), depth 100.

```
[coin icon] 12        Day 1
```

| Element | X | Y | Details |
|---------|---|---|---------|
| Coin icon | 16px | 16px | 16x16 gold circle or coin sprite, accent color |
| Coin count | 40px | 16px | 20px bold, accent color (#e2b714) |
| Day label | 100px | 16px | 14px regular, white (#FFFFFF) "Day" |
| Day number | 130px | 16px | 20px bold, white (#FFFFFF) |

### Behavior

- Coin count updates in real time when coins are collected, spent, or earned from farms
- Day number is static at "1" for Phase 2 (day/night cycle is Phase 3); placeholder display only
- No animation on count change for Phase 2 (optional: brief scale pulse on coin gain — Claude's discretion)
- Replaces the Phase 1 debug `coinCountText` entirely (D-14)

## Interaction Contracts

### Vagrant Recruitment (D-01, D-02)

| Step | Player Action | Visual Feedback |
|------|--------------|-----------------|
| 1 | Hero walks near a vagrant (within 64px) | Vagrant stops drifting, faces hero |
| 2 | Hero presses drop key (Space/Down) | Coin flies from hero to vagrant (200ms tween) |
| 3 | Coin arrives | Vagrant flashes white (100ms), color shifts to citizen blue (#4488CC) |
| 4 | Conversion complete | New citizen runs toward main base at 120px/sec |

- Cost: 1 coin per vagrant
- If hero has 0 coins: hero red flash (existing Phase 1 feedback), vagrant unchanged

### Coin Drop at Build Point (extended from Phase 1)

| Step | Player Action | Visual Feedback |
|------|--------------|-----------------|
| 1 | Hero walks near build point (64px radius) | Build point pulse intensifies (Phase 1 pattern) |
| 2 | Hero presses drop key | Coin flies from hero to build point (200ms tween) |
| 3 | Coin threshold reached | Structure begins construction — builder walks to site |
| 4 | Builder arrives | Hammering animation (Y oscillation), structure rectangle grows from 0 height to full height over build duration |
| 5 | Construction complete | Structure at full size, build point marker hidden |

### Structure Upgrade

| Step | Player Action | Visual Feedback |
|------|--------------|-----------------|
| 1 | Hero walks near existing structure's build point | Build point pulse reappears around structure base |
| 2 | Hero drops coins (upgrade cost threshold) | Builder walks to structure, hammering animation |
| 3 | Upgrade complete | Structure grows (height/width increase), color shifts, brief white flash + scale pulse |

### Main Base Upgrade (D-07)

| Upgrade | Cost | Visual Change | Gameplay Unlock |
|---------|------|--------------|-----------------|
| Tier 1 -> 2 | 5 coins | Base grows wider (96->128px), wings appear, darker color | Builder hut + tower build points become visible |
| Tier 2 -> 3 | 15 coins | Base grows wider (128->160px), wings extend, darkest color | Farm build points become visible |
| Tier 3 -> max | 30 coins | Base grows to max (160->192px), HP increase | Claude's discretion — visual crown/flag indicator |

### Main Base Destruction (D-09, STRC-03)

| Step | Trigger | Visual Feedback |
|------|---------|-----------------|
| 1 | Base HP reaches 0 | All entities freeze (pause physics) |
| 2 | 0ms | Base rectangle shakes (x oscillation, 3px, 100ms, 5 repeats) |
| 3 | 500ms | Base splits into 4-6 smaller rectangles that fall with gravity (brief particle-like collapse) |
| 4 | 1500ms | Screen flashes red (100ms), fade to black (500ms) |
| 5 | 2000ms | Transition to GameOver scene |

## Interaction Feedback Summary

| Event | Visual Feedback | Audio |
|-------|----------------|-------|
| Coin collected (by hero) | Coin tweens to hero, fades (Phase 1 pattern) | None |
| Coin earned (from farm) | Gold "+1" text floats up from farm, fades over 800ms | None |
| Coin spent (drop at build point) | Coin flies to target (200ms tween) | None |
| Coin spent (recruit vagrant) | Coin flies to vagrant (200ms tween) | None |
| Drop attempt, 0 coins | Hero red flash (100ms) — Phase 1 pattern | None |
| Vagrant recruited | White flash on vagrant, color shift to blue | None |
| Structure construction starts | Builder walks to site, hammering tween begins | None |
| Structure construction completes | Structure snaps to full size, brief scale pulse (1.0->1.1->1.0, 200ms) | None |
| Structure upgrade completes | Size/color change, white flash (100ms), scale pulse | None |
| Main base upgrade | Wings extend, color darkens, new build points fade in (alpha 0->0.5, 500ms) | None |
| Main base damaged | Health bar appears, structure tints red briefly (50ms) | None |
| Main base destroyed | Shake -> collapse particles -> red flash -> fade to black -> GameOver | None |
| Near build point | Pulse intensifies (Phase 1 pattern) | None |
| Build point unlocked | Fade in from alpha 0 to idle pulse range over 500ms | None |

## Copywriting Contract

| Element | Copy |
|---------|------|
| HUD coin display | `[coin icon] {count}` — no label, icon is sufficient |
| HUD day display | `Day {number}` — "Day 1" for Phase 2 (static until Phase 3) |
| Build point cost indicator | `{cost}` — gold number floating above unbuilt build points when hero is nearby |
| Structure upgrade cost | `+{cost}` — gold number above upgradeable structures when hero is nearby |
| Farm income indicator | `+1` — gold text floating up from farm on each coin generation tick |
| Vagrant recruitment prompt | No text — the coin-drop mechanic is the prompt (consistent with "no menus" philosophy) |
| Empty state (no citizens) | Not applicable — vagrants always exist in the world for recruitment |
| Empty state (no coins) | No explicit message — HUD shows "0", hero red flash on attempted drop is sufficient feedback |
| Error state (cannot build) | Not applicable — build points only appear when unlocked and accept coins silently |
| Destructive: base destroyed | No confirmation — this is an enemy-triggered event, not a player action |
| GameOver screen | "YOUR CITY HAS FALLEN" (24px, bold, white, centered) + "Press ENTER to restart" (14px, regular, white) |
