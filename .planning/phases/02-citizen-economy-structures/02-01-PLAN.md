**Step 2: Create `src/systems/EconomyManager.ts`**

Create the `src/systems/` directory. EconomyManager centralizes coin count and day number. ALL coin changes go through this class. It emits `economy:changed` on EventBus whenever coins or day change.

```typescript
import EventBus from '../events/EventBus';

export class EconomyManager {
  private _coins: number = 0;
  private _day: number = 1;

  get coins(): number { return this._coins; }
  get day(): number { return this._day; }

  addCoins(amount: number, source: string): void {
    this._coins += amount;
    EventBus.emit('economy:changed', { coins: this._coins, day: this._day, source });
  }

  spendCoins(amount: number): boolean {
    if (this._coins < amount) return false;
    this._coins -= amount;
    EventBus.emit('economy:changed', { coins: this._coins, day: this._day, source: 'spend' });
    return true;
  }

  setDay(day: number): void {
    this._day = day;
    EventBus.emit('economy:changed', { coins: this._coins, day: this._day, source: 'day-change' });
  }

  /** Reset economy state (for scene restart) */
  reset(): void {
    this._coins = 0;
    this._day = 1;
  }
}
```

Key design decisions:
- `addCoins` takes a `source` string for debugging/tracking
- `spendCoins` returns boolean (false if insufficient) so callers can guard
- `reset()` for scene restart without creating a new instance
- No scene dependency -- pure data class that emits through EventBus

The HUD displays coin count and day number at top-left, fixed to camera. It listens to `economy:changed` EventBus events to stay in sync with EconomyManager.

Per D-14, this replaces the Phase 1 debug `coinCountText`. Per D-15, scope is coin count + day number only.

```typescript
import Phaser from 'phaser';
import EventBus from '../events/EventBus';
import {
  HUD_MARGIN, HUD_COIN_ICON_SIZE, HUD_COIN_TEXT_SIZE,
  HUD_LABEL_TEXT_SIZE, HUD_COIN_X, HUD_DAY_LABEL_X,
  HUD_Y, HUD_DEPTH, COLOR_ACCENT
} from '../constants';

export class HUD {
  private coinIcon: Phaser.GameObjects.Arc;
  private coinText: Phaser.GameObjects.Text;
  private dayText: Phaser.GameObjects.Text;
  private onEconomyChanged: (data: { coins: number; day: number }) => void;

  constructor(scene: Phaser.Scene) {
    // Coin icon -- small gold circle at top-left
    this.coinIcon = scene.add.circle(
      HUD_MARGIN, HUD_Y, HUD_COIN_ICON_SIZE / 2, COLOR_ACCENT
    ).setScrollFactor(0).setDepth(HUD_DEPTH);

    // Coin count text -- bold, accent color, right of icon
    this.coinText = scene.add.text(HUD_COIN_X, HUD_Y, '0', {
      fontFamily: '"Courier New", monospace',
      fontSize: HUD_COIN_TEXT_SIZE,
      color: '#e2b714',
      fontStyle: 'bold',
    }).setScrollFactor(0).setDepth(HUD_DEPTH).setOrigin(0, 0.3);

    // Day text -- "Day 1", white, right of coin count
    this.dayText = scene.add.text(HUD_DAY_LABEL_X, HUD_Y, 'Day 1', {
      fontFamily: '"Courier New", monospace',
      fontSize: HUD_LABEL_TEXT_SIZE,
      color: '#FFFFFF',
    }).setScrollFactor(0).setDepth(HUD_DEPTH).setOrigin(0, 0.3);

    // Listen for economy changes
    this.onEconomyChanged = (data: { coins: number; day: number }) => {
      this.coinText.setText(`${data.coins}`);
      this.dayText.setText(`Day ${data.day}`);
    };
    EventBus.on('economy:changed', this.onEconomyChanged);
  }

  /** Remove EventBus listeners and destroy game objects */
  destroy(): void {
    EventBus.off('economy:changed', this.onEconomyChanged);
    this.coinIcon.destroy();
    this.coinText.destroy();
    this.dayText.destroy();
  }
}
```

Key details:
- Store the listener function reference so it can be removed in `destroy()` (avoids listener leak per Pitfall 1 from RESEARCH.md)
- `setOrigin(0, 0.3)` vertically centers text relative to the coin icon circle
- All elements use `setScrollFactor(0)` and `setDepth(HUD_DEPTH)` (100) to stay fixed on camera above all game objects
- Per D-16, day counter uses simple "Day N" text format
