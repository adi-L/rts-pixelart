**Create `src/entities/Coin.ts`** -- helper functions for the coin pool (not a class, since pool management lives in Game.ts per RESEARCH.md pattern):

```typescript
import Phaser from 'phaser';
import {
  COIN_SIZE, COIN_SCALE, COLOR_ACCENT, COIN_SPIN_DURATION,
  COIN_COLLECT_DURATION, GROUND_Y, SPRITE_COIN
} from '../constants';
```

1. **`createCoinTexture(scene: Phaser.Scene): void`** -- Generate a placeholder coin texture programmatically (no external asset needed per D-11):
   - Use `scene.add.graphics()` to draw a filled circle: `graphics.fillStyle(COLOR_ACCENT, 1); graphics.fillCircle(COIN_SIZE / 2, COIN_SIZE / 2, COIN_SIZE / 2);`
   - Generate texture: `graphics.generateTexture(SPRITE_COIN, COIN_SIZE, COIN_SIZE);`
   - Destroy the graphics object: `graphics.destroy();`

2. **`createCoinPool(scene: Phaser.Scene): Phaser.Physics.Arcade.Group`** -- Create the object pool per INFRA-02:
   ```typescript
   const pool = scene.physics.add.group({
     classType: Phaser.Physics.Arcade.Sprite,
     maxSize: COIN_POOL_SIZE,
     runChildUpdate: false,
     allowGravity: false,
   });
   pool.createMultiple({
     key: SPRITE_COIN,
     quantity: COIN_POOL_SIZE,
     active: false,
     visible: false,
   });
   return pool;
   ```

3. **`spawnCoin(pool: Phaser.Physics.Arcade.Group, x: number, y: number, scene: Phaser.Scene): Phaser.Physics.Arcade.Sprite | null`** -- Activate a coin from pool:
   - `const coin = pool.get(x, y, SPRITE_COIN) as Phaser.Physics.Arcade.Sprite;`
   - Null-check (Pitfall 4 in RESEARCH.md): `if (!coin) return null;`
   - `coin.setActive(true).setVisible(true).setScale(COIN_SCALE);`
   - Enable physics body: `if (coin.body) { (coin.body as Phaser.Physics.Arcade.Body).enable = true; }`
   - Add spin tween (D-11 "spinning"): `scene.tweens.add({ targets: coin, scaleX: { from: COIN_SCALE, to: 0 }, duration: COIN_SPIN_DURATION / 2, yoyo: true, repeat: -1 });`
   - Return coin

4. **`collectCoin(coin: Phaser.Physics.Arcade.Sprite, heroSprite: Phaser.Physics.Arcade.Sprite, scene: Phaser.Scene): void`** -- Magnetic collection per D-09:
   - Immediately disable physics to prevent double-collect: `if (coin.body) { (coin.body as Phaser.Physics.Arcade.Body).enable = false; }`
   - Tween coin to hero position: `scene.tweens.add({ targets: coin, x: heroSprite.x, y: heroSprite.y, alpha: 0, duration: COIN_COLLECT_DURATION, onComplete: () => { coin.setActive(false).setVisible(false).setAlpha(1); scene.tweens.killTweensOf(coin); } });`

Export all four functions as named exports.

**Create `src/entities/BuildPoint.ts`** -- class for build point markers per D-12:

```typescript
import Phaser from 'phaser';
import EventBus from '../events/EventBus';
import {
  BUILD_POINT_WIDTH, BUILD_POINT_HEIGHT, BUILD_POINT_DETECT_RADIUS,
  BUILD_POINT_IDLE_ALPHA_MIN, BUILD_POINT_IDLE_ALPHA_MAX,
  BUILD_POINT_ACTIVE_ALPHA_MIN, BUILD_POINT_ACTIVE_ALPHA_MAX,
  BUILD_POINT_PULSE_DURATION, COLOR_ACCENT, GROUND_Y
} from '../constants';
```

The class:

1. **Properties:**
   - `id: string` -- unique identifier (e.g., 'bp-0', 'bp-1')
   - `x: number` -- world x position
   - `marker: Phaser.GameObjects.Rectangle` -- the visual marker
   - `pulseTween: Phaser.Tweens.Tween` -- the pulsing animation
   - `coinsDeposited: number` -- running total of coins deposited here
   - `isNearby: boolean` -- whether hero is within detection radius

2. **constructor(scene: Phaser.Scene, x: number, id: string):**
   - Store scene, x, id
   - Create marker rectangle at (x, GROUND_Y - BUILD_POINT_HEIGHT / 2): `this.marker = scene.add.rectangle(x, GROUND_Y - BUILD_POINT_HEIGHT / 2, BUILD_POINT_WIDTH, BUILD_POINT_HEIGHT, COLOR_ACCENT).setAlpha(BUILD_POINT_IDLE_ALPHA_MIN).setDepth(2);`
   - Start idle pulse tween: `this.pulseTween = scene.tweens.add({ targets: this.marker, alpha: { from: BUILD_POINT_IDLE_ALPHA_MIN, to: BUILD_POINT_IDLE_ALPHA_MAX }, duration: BUILD_POINT_PULSE_DURATION, yoyo: true, repeat: -1 });`
   - `this.coinsDeposited = 0;`
   - `this.isNearby = false;`

3. **setNearby(near: boolean):**
   - If state unchanged, return early
   - `this.isNearby = near;`
   - Stop current pulse tween: `this.pulseTween.stop();`
   - If near: start active pulse `{ from: BUILD_POINT_ACTIVE_ALPHA_MIN, to: BUILD_POINT_ACTIVE_ALPHA_MAX }`
   - If not near: start idle pulse `{ from: BUILD_POINT_IDLE_ALPHA_MIN, to: BUILD_POINT_IDLE_ALPHA_MAX }`

4. **addCoin(): number:**
   - `this.coinsDeposited++;`
   - `EventBus.emit('coin:deposited', { buildPointId: this.id, total: this.coinsDeposited });`
   - Return `this.coinsDeposited`

5. **destroy():**
   - `this.pulseTween.stop();`
   - `this.marker.destroy();`

Export as named export: `export class BuildPoint { ... }`
