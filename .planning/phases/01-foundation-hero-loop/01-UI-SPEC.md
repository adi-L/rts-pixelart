# Phase 1 — UI Design Contract

> Visual and interaction contract for the Foundation & Hero Loop phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Note:** This is a Phaser 3 canvas game. All visual values are in-engine constants (pixels, hex colors, tween configs), not CSS/DOM tokens. There is no web UI framework, no shadcn, no Tailwind.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Phaser 3 canvas rendering) |
| Preset | not applicable |
| Component library | none (Phaser GameObjects) |
| Icon library | none (pixel art sprites) |
| Font | Phaser bitmap text or system monospace for debug; no custom font in Phase 1 |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Sprite padding, collision body insets |
| sm | 8px | Gap between coin sprites, build point marker padding |
| md | 16px | Default spacing between world elements |
| lg | 24px | Hero-to-ground offset, coin float height |
| xl | 32px | Build point detection radius (horizontal) |
| 2xl | 48px | Hero sprite frame size (48x48 target) |
| 3xl | 64px | Minimum distance between build points |

Exceptions: Touch target not applicable (keyboard-only input in Phase 1).

---

## Typography

Phase 1 has no HUD text (HUD is Phase 2). Text is limited to debug overlays and scene transitions.

| Role | Size | Weight | Line Height | Context |
|------|------|--------|-------------|---------|
| Debug overlay | 12px | 400 (monospace) | 1.0 | FPS counter, entity count — dev only |
| Scene title | 24px | 700 | 1.2 | MainMenu scene "Dead City" title text |
| Scene subtitle | 14px | 400 | 1.4 | MainMenu "Press any key" prompt |

Font family for all text: `"Courier New", monospace` (Phaser Text style). No custom font loading in Phase 1. Phase 2 will introduce a pixel font for the HUD.

## Color

All colors are hex values used in Phaser `setTint()`, `fillStyle`, or `fillColor`.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #1a1a2e | Sky/background tint — dark blue-purple, post-apocalyptic night feel |
| Secondary (30%) | #16213e | Mid-ground parallax layer tint, ground fill |
| Accent (10%) | #e2b714 | Coin glow, build point pulse, pickup feedback flash |
| Ground surface | #3a3a3a | Ground rectangle fill color |
| Ground top edge | #5a5a5a | 2px highlight line at ground surface for depth |

Accent reserved for:
- Coin sprite tint/glow
- Build point pulsing marker
- Coin collection feedback flash (brief white-to-gold tween)

No destructive color needed in Phase 1 (no damage, no death, no delete actions).

## Sprite Dimensions & Animation

### Hero

| Property | Value | Source |
|----------|-------|--------|
| Spritesheet | `gunner/Tiny Gunslinger 48x32.png` or `gunner/lizzy.png` | D-08 |
| Frame size | 48x32px (Tiny Gunslinger) | Asset file name |
| Display scale | 2x (renders at 96x64 on screen) | Pixel art crispness at 768p |
| Idle animation | 4 frames, 8 fps | Claude's discretion |
| Walk animation | 6 frames, 10 fps | Claude's discretion |
| Direction flip | `sprite.setFlipX(true)` for left-facing | D-06 |
| Physics body | 24px wide x 48px tall, centered on sprite | Narrower than visual for forgiving collisions |
| Movement speed | 300 px/sec (~20 seconds to cross 6000px world) | D-07 |

### Coin

| Property | Value | Source |
|----------|-------|--------|
| Sprite | Placeholder circle/rectangle, 12x12px | D-11 |
| Display scale | 2x (renders at 24x24 on screen) | Visibility at distance |
| Tint | #e2b714 (gold accent) | Color contract |
| Spin animation | Tween: `scaleX` from 1 to 0 to 1, duration 600ms, loop -1 | D-11 "spinning" |
| Collection effect | Tween to hero position over 150ms + fade out | D-09 "magnetic feel" |
| Ground offset | Resting at groundY - 12px (half sprite height above ground) | Natural placement |

### Build Point

| Property | Value | Source |
|----------|-------|--------|
| Visual | Vertical dashed line or glowing rectangle, 32px wide x 48px tall | D-12 |
| Tint | #e2b714 at 40% alpha | Color contract, subtle when inactive |
| Pulse animation | Alpha tween: 0.3 to 0.7, duration 1000ms, yoyo, loop -1 | D-12 "pulsing" |
| Active state | Alpha tween: 0.7 to 1.0 when hero is within detection radius | Feedback |
| Detection radius | 32px horizontal from center | Spacing scale xl |
| Coin drop position | Center of build point, groundY | Logical target |

## Parallax Layers

3 layers per D-03. All TileSprite, full canvas width, scrolling via camera scroll factor.

| Layer | Depth | Scroll Factor | Content | Color/Tint |
|-------|-------|---------------|---------|------------|
| Sky (back) | 0 | 0.0 (static) | Solid gradient or static image | #1a1a2e to #0f3460 vertical gradient |
| Mid-ground | 1 | 0.3 | Silhouette buildings/trees | #16213e, low detail shapes |
| Ground | 2 | 1.0 (locked to world) | Ground platform rectangle | #3a3a3a with #5a5a5a top edge |
| Entities | 3 | 1.0 | Hero, coins, build points | Per-entity colors above |

## Input Mapping

| Action | Key | Behavior | Source |
|--------|-----|----------|--------|
| Move left | Arrow Left / A | Set hero velocity -300 | HERO-01 |
| Move right | Arrow Right / D | Set hero velocity +300 | HERO-01 |
| Stop | Release movement key | Set hero velocity 0, switch to idle | HERO-01 |
| Drop coin | Spacebar / Down Arrow | Drop 1 coin if near build point AND coin count > 0 | D-10 |

No jumping (D-05). No mouse input in Phase 1.

---

## Interaction Feedback

| Event | Visual Feedback | Audio |
|-------|----------------|-------|
| Coin collected | Coin tweens toward hero over 150ms, fades out | None (Phase 1 has no audio) |
| Coin dropped | Coin sprite appears at build point, brief scale-up tween (1.0 to 1.3 to 1.0, 200ms) | None |
| Near build point | Build point alpha increases from 0.3-0.7 range to 0.7-1.0 range | None |
| Leave build point | Build point alpha returns to idle pulse range | None |
| Drop attempt with 0 coins | Brief red tint flash on hero (100ms) | None |
| Drop attempt far from build point | No feedback (button press ignored silently) | None |

## State Indicators

| State | Visual Representation |
|-------|-----------------------|
| Hero idle | Idle animation loop (4 frames) |
| Hero walking | Walk animation loop (6 frames) |
| Hero facing direction | Sprite flipX (false = right, true = left) |
| Coin in world (uncollected) | Spinning tween on ground, gold tint |
| Coin collected | Tween to hero + fade (removed from world, returned to pool) |
| Build point idle | Slow alpha pulse (0.3-0.7), gold tint at 40% alpha |
| Build point active (hero near) | Faster alpha pulse (0.7-1.0), gold tint at 70% alpha |
| Coin deposited at build point | EventBus fires `coin:deposited` with build point ID and running total |
