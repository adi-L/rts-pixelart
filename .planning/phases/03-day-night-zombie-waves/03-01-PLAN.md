**Create `src/systems/DayNightCycleManager.ts`:**

A system class with these responsibilities:
- Tracks elapsed time in the current phase (day or night)
- Manages a full-screen `Phaser.GameObjects.Rectangle` overlay at `OVERLAY_DEPTH` (50), `setScrollFactor(0)`, starts at alpha 0
- The overlay is positioned at `scene.scale.width / 2, scene.scale.height / 2` with dimensions `scene.scale.width, scene.scale.height`
- On each `update(delta)` call, increments `this.elapsed` by `delta`
- When in day phase and elapsed reaches `DAY_DURATION - NIGHTFALL_WARNING_TIME`: emit `EventBus.emit('night:warning')`, set `warningEmitted = true`
- When elapsed reaches the current phase duration:
  - Reset elapsed to 0, toggle `isNight`
  - If transitioning to night: call `transitionToNight()`
  - If transitioning to day: call `transitionToDay()`
- `transitionToNight()`: tween overlay alpha from 0 to `OVERLAY_MAX_ALPHA` over `TRANSITION_DURATION` ms with `Sine.easeInOut` easing. On complete: `EventBus.emit('night:start')`
- `transitionToDay()`: call `this.economy.setDay(this.economy.day + 1)` to increment day (per D-04). Emit `EventBus.emit('night:end')`. Tween overlay alpha from current to 0 over `TRANSITION_DURATION` ms. On complete: `EventBus.emit('day:start')`
- `destroy()`: kill tweens on overlay, destroy overlay game object

Constructor signature: `constructor(scene: Phaser.Scene, economy: EconomyManager)`

Store `scene` and `economy` as private fields. Initialize: `elapsed = 0`, `isNight = false`, `warningEmitted = false`.

Import from constants: `DAY_DURATION, NIGHT_DURATION, TRANSITION_DURATION, NIGHTFALL_WARNING_TIME, OVERLAY_NIGHT_COLOR, OVERLAY_MAX_ALPHA, OVERLAY_DEPTH`

Public getter `get isNightTime(): boolean` for other systems to query.

```typescript
this.onNightWarning = () => {
  // Flash warning text 3 times over 3 seconds then fade out
  scene.tweens.add({
    targets: this.warningText,
    alpha: { from: 0, to: 1 },
    duration: 500,
    yoyo: true,
    repeat: 2,
    onComplete: () => this.warningText.setAlpha(0),
  });
};
EventBus.on('night:warning', this.onNightWarning);
```
5. In `destroy()`, add:
```typescript

this.warningText.destroy();
```
