Create `src/entities/Arrow.ts` following the exact pattern of `Coin.ts` (texture gen, pool creation, spawn, deactivate):

**`createArrowPool(scene: Phaser.Scene): Phaser.Physics.Arcade.Group`**:
```typescript
const pool = scene.physics.add.group({
  classType: Phaser.Physics.Arcade.Sprite,
  maxSize: ARROW_POOL_SIZE,
  runChildUpdate: false,
  allowGravity: false,
});
pool.createMultiple({
  key: SPRITE_ARROW,
  quantity: ARROW_POOL_SIZE,
  active: false,
  visible: false,
});
return pool;
```

**`fireArrow(pool: Phaser.Physics.Arcade.Group, fromX: number, fromY: number, targetX: number, targetY: number, scene: Phaser.Scene): Phaser.Physics.Arcade.Sprite | null`**:
- Get from pool: `const arrow = pool.get(fromX, fromY, SPRITE_ARROW) as Phaser.Physics.Arcade.Sprite`
- If null, return null (pool exhausted)
- Activate: `arrow.setActive(true).setVisible(true).setDepth(5)`
- Enable physics: `scene.physics.world.enable(arrow); (arrow.body as Phaser.Physics.Arcade.Body).enable = true`
- Calculate velocity: `const angle = Phaser.Math.Angle.Between(fromX, fromY, targetX, targetY); arrow.body.setVelocity(Math.cos(angle) * ARROW_SPEED, Math.sin(angle) * ARROW_SPEED)`
- Store start position: `arrow.setData('startX', fromX); arrow.setData('startY', fromY)`
- Return arrow

**`deactivateArrow(arrow: Phaser.Physics.Arcade.Sprite): void`**:
- `arrow.setActive(false).setVisible(false)`
- `if (arrow.body) { (arrow.body as Phaser.Physics.Arcade.Body).enable = false; (arrow.body as Phaser.Physics.Arcade.Body).setVelocity(0, 0); }`

**`updateArrows(pool: Phaser.Physics.Arcade.Group): void`**:
- Iterate active children of the pool
- For each active arrow, check if it has traveled beyond `ARROW_MAX_RANGE` from its start position: `const dx = arrow.x - arrow.getData('startX'); const dy = arrow.y - arrow.getData('startY'); if (Math.sqrt(dx*dx + dy*dy) > ARROW_MAX_RANGE) deactivateArrow(arrow)`
- Also check world bounds: if `arrow.x < 0 || arrow.x > WORLD_WIDTH`, deactivate

**Create `src/entities/npc/Archer.ts`:**

Extends BaseNPC with a three-state machine driven by EventBus events.

```typescript
export enum ArcherState {
  Hunt,
  Returning,
  Defend,
}
```

**Constructor** `(scene: Phaser.Scene, x: number, towerX: number, arrowPool: Phaser.Physics.Arcade.Group, economy: EconomyManager, zombiePoolGetter: () => Phaser.Physics.Arcade.Group)`:
- `super(scene, x, COLOR_ARCHER)` -- creates the orange-brown rectangle NPC
- Store `towerX`, `arrowPool`, `economy`, `zombiePoolGetter` as private fields
- Initialize `state = ArcherState.Hunt`
- Initialize `lastFireTime = 0`, `lastHuntCoinTime = 0`, `wanderTarget = towerX`
- Register EventBus listeners (store references for destroy cleanup):
  - `this.onNightWarning = () => { this.state = ArcherState.Returning; }` -- on `night:warning`
  - `this.onNightStart = () => { this.state = ArcherState.Defend; this.sprite.body.setVelocityX(0); this.sprite.setPosition(this.towerX, this.sprite.y); }` -- on `night:start`, snap to tower
  - `this.onDayStart = () => { this.state = ArcherState.Hunt; this.wanderTarget = this.towerX; }` -- on `day:start`
  - `this.onTowerDestroyed = (data: { buildPointId: string }) => { /* handled by Game.ts -- archer checks if its tower still exists */ }` -- listen for `structure:destroyed` if emitted, or check tower HP in update
- `EventBus.on('night:warning', this.onNightWarning)`
- `EventBus.on('night:start', this.onNightStart)`
- `EventBus.on('day:start', this.onDayStart)`

**`update(time: number, delta: number)`**:
- `if (this.destroyed) return;`
- Switch on `this.state`:
  - **Hunt**: Call `this.huntBehavior(time, delta)`
    - Wander near tower: if arrived at `wanderTarget`, pick new random target within `towerX +/- ARCHER_WANDER_DISTANCE`
    - `this.moveToward(this.wanderTarget, FARMER_WALK_SPEED)` (use farmer speed for casual walking, import `FARMER_WALK_SPEED` from constants)
    - Check hunt coin timer: if `time - this.lastHuntCoinTime >= ARCHER_HUNT_COIN_INTERVAL`, generate 1 coin: `this.economy.addCoins(1, 'archer-hunt')`, update `lastHuntCoinTime = time`
  - **Returning**: `if (this.moveToward(this.towerX, ARCHER_RETURN_SPEED)) { this.state = ArcherState.Defend; }`
  - **Defend**: Call `this.defendBehavior(time)`
    - Stay at tower position (no movement)
    - Scan for nearest active zombie in `zombiePoolGetter()` children
    - Find closest active zombie sprite within `ARCHER_RANGE` of `this.towerX`
    - If found and `time - this.lastFireTime >= ARCHER_FIRE_RATE`:
      - `fireArrow(this.arrowPool, this.sprite.x, this.sprite.y - NPC_HEIGHT / 2, zombie.x, zombie.y, this.scene)`
      - `this.lastFireTime = time`

**`destroy()`**:
- `EventBus.off('night:warning', this.onNightWarning)`
- `EventBus.off('night:start', this.onNightStart)`
- `EventBus.off('day:start', this.onDayStart)`
- `super.destroy()`
