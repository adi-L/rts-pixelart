  <action>
Create `src/systems/WaveManager.ts` with these responsibilities:

**Constructor** `(scene: Phaser.Scene, structureManager: StructureManager, npcManager: NPCManager)`:
- Store scene, structureManager, npcManager as private fields
- Create zombie pool: `scene.physics.add.group({ classType: Phaser.Physics.Arcade.Sprite, maxSize: ZOMBIE_POOL_SIZE, runChildUpdate: false, allowGravity: false })`
- Pre-create pool: `this.zombiePool.createMultiple({ key: SPRITE_ZOMBIE, quantity: ZOMBIE_POOL_SIZE, active: false, visible: false })`
- Store `currentNight = 0`
- Register EventBus listeners (store references for cleanup):
  - `EventBus.on('night:start', this.onNightStart)` -- calls `startWave()`
  - `EventBus.on('night:end', this.onNightEnd)` -- calls `killAllZombies()`
- Private field `spawnTimer: Phaser.Time.TimerEvent | null = null`
- Private field `activeZombies: Phaser.Physics.Arcade.Sprite[] = []` to track active zombies for update loop

**Public getter** `get zombiePool(): Phaser.Physics.Arcade.Group` -- needed by Game.ts to register overlaps

**`startWave()`**:
- Increment `this.currentNight`
- Calculate total zombies: `ZOMBIE_BASE_COUNT + (this.currentNight - 1) * ZOMBIE_GROWTH_PER_NIGHT` (per D-16)
- Calculate spawn interval: `(NIGHT_DURATION * 0.8) / totalZombies` -- spread over 80% of night (per D-11)
- Create a staggered spawn timer:
```typescript
let spawned = 0;
this.spawnTimer = this.scene.time.addEvent({
  delay: interval,
  repeat: totalZombies - 1,
  callback: () => {
    const x = spawned % 2 === 0 ? ZOMBIE_SPAWN_MARGIN : WORLD_WIDTH - ZOMBIE_SPAWN_MARGIN;
    this.spawnZombie(x);
    spawned++;
  },
});
```

**`spawnZombie(x: number)`**:
- Get sprite from pool: `const zombie = this.zombiePool.get(x, GROUND_Y - NPC_HEIGHT / 2, SPRITE_ZOMBIE)`
- If null, `console.warn('Zombie pool exhausted')` and return
- Activate: `zombie.setActive(true).setVisible(true).setDepth(3)`
- Enable physics: `this.scene.physics.world.enable(zombie); (zombie.body as Phaser.Physics.Arcade.Body).enable = true; zombie.body.setAllowGravity(false)`
- Set size to match NPC dimensions: `zombie.setDisplaySize(NPC_WIDTH, NPC_HEIGHT)`
- Initialize data: `zombie.setData('state', ZombieState.March)`, `zombie.setData('hp', ZOMBIE_HP)`, `zombie.setData('lastAttack', 0)`
- Determine march direction: if x < WORLD_WIDTH / 2, velocity = ZOMBIE_SPEED, else velocity = -ZOMBIE_SPEED
- Set velocity: `(zombie.body as Phaser.Physics.Arcade.Body).setVelocityX(velocity)`
- Push to `this.activeZombies`

**`update(time: number, delta: number)`**:
- Iterate `this.activeZombies` (backwards for safe removal):
  - Skip if `!zombie.active`
  - Get state from `zombie.getData('state')`
  - If `ZombieState.March`: find nearest alive structure using `findNearestStructure(zombie.x)`. If found, set velocity toward it. If no structures remain, march toward `WORLD_WIDTH / 2` (the base position).
  - If `ZombieState.Attack`: velocity is already 0, attack timer is handled by overlap callback. Check if the structure being attacked is destroyed -- if so, switch back to `ZombieState.March` and set velocity again.
  - If `ZombieState.Dying`: skip (handled by tween)

**`findNearestStructure(x: number): BaseStructure | null`**:
- Search `this.structureManager.structures` plus `this.structureManager.mainBase`
- Filter out destroyed structures (`isDestroyed`)
- Return the one with minimum `Math.abs(structure.x - x)`

**`handleZombieStructureOverlap(zombie: Phaser.Physics.Arcade.Sprite, structureRect: Phaser.GameObjects.Rectangle)`**:
- Public method called from Game.ts overlap callback
- If `!zombie.active` return
- If `zombie.getData('state') === ZombieState.Dying` return
- Stop zombie movement: `(zombie.body as Phaser.Physics.Arcade.Body).setVelocityX(0)`
- Set state to Attack: `zombie.setData('state', ZombieState.Attack)`
- Check cooldown: `const now = this.scene.time.now; const last = zombie.getData('lastAttack') || 0; if (now - last < ZOMBIE_ATTACK_INTERVAL) return;`
- Set last attack time: `zombie.setData('lastAttack', now)`
- Find the structure by matching `structureRect`: iterate `this.structureManager.structures` and check if `s.rect === structureRect`, also check `this.structureManager.mainBase.rect === structureRect`
- If found and not destroyed: `structure.takeDamage(ZOMBIE_DAMAGE)`
- If structure is now destroyed: switch zombie back to March state

**`handleZombieNPCOverlap(zombie: Phaser.Physics.Arcade.Sprite, npcSprite: Phaser.GameObjects.Rectangle)`**:
- Public method called from Game.ts overlap callback (per D-08)
- If `!zombie.active` return
- If `zombie.getData('state') === ZombieState.Dying` return
- Find the NPC in npcManager arrays (vagrants, citizens, builders, farmers) whose `sprite === npcSprite`
- If found: remove from the respective array (splice), call `npc.destroy()`
- Zombie continues marching (does not stop for NPCs)

**`damageZombie(zombie: Phaser.Physics.Arcade.Sprite, damage: number)`**:
- Public method called from arrow overlap in Game.ts
- `const hp = zombie.getData('hp') - damage; zombie.setData('hp', hp);`
- If hp <= 0: call `this.killZombie(zombie)`

**`killZombie(zombie: Phaser.Physics.Arcade.Sprite)`**:
- Set state to Dying: `zombie.setData('state', ZombieState.Dying)`
- Stop movement: `(zombie.body as Phaser.Physics.Arcade.Body).setVelocityX(0)`
- Fade out tween: `this.scene.tweens.add({ targets: zombie, alpha: 0, duration: 300, onComplete: () => this.deactivateZombie(zombie) })`

**`deactivateZombie(zombie: Phaser.Physics.Arcade.Sprite)`**:
- `zombie.setActive(false).setVisible(false).setAlpha(1)`
- `(zombie.body as Phaser.Physics.Arcade.Body).enable = false`
- `(zombie.body as Phaser.Physics.Arcade.Body).setVelocity(0, 0)`
- Remove from `this.activeZombies` array

**`killAllZombies()`** (called on `night:end` per D-09):
- For each zombie in `activeZombies`: fade out tween then deactivate (same as `killZombie` but can batch)
- Clear spawn timer if active: `if (this.spawnTimer) { this.spawnTimer.destroy(); this.spawnTimer = null; }`

**`destroy()`**:
- `EventBus.off('night:start', this.onNightStart)`
- `EventBus.off('night:end', this.onNightEnd)`
- `if (this.spawnTimer) this.spawnTimer.destroy()`
- Kill all tweens on active zombies
- Deactivate all zombies
- `this.zombiePool.destroy(true)`
